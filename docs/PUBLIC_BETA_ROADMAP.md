# SkillMint 50-User Beta Roadmap

## Position

This is a controlled 50-user beta, not a full public launch. Do not expand features if the core journey fails.

## Target Users

- Final-year students
- Freshers
- Internship seekers
- 0-2 year early-career candidates
- Indian engineering, tech, and business students first

## Timeline

- Day 1-3: Product Trust + Beta Readiness block.
- Day 4: Review, deploy, and smoke test.
- Day 5-7: 5-user close pilot.
- Day 8-10: Fix only P0/P1 blockers and serious confusion.
- Day 11-17: Invite 20-30 users in batches.
- Day 18-21: Reach 50 invited users and collect feedback.

## Success Metrics

- 50 invited users.
- 30+ signups.
- 20+ resume uploads.
- 15+ dashboard views after upload.
- 10+ JD matches.
- 10+ feedback responses.
- 0 unresolved P0/P1 bugs.
- At least 60% understand what SkillMint does.
- At least 40% say they would use it again.

## Failure Signals

- Scoring feels fake.
- UI feels tacky or untrustworthy.
- Users cannot explain Career IQ.
- Users cannot tell profile-fit role from JD match.
- Users cannot tell saved report from active report.
- Upload fails.
- Mobile blocks completion.

## Go / No-Go Rule

Do not expand features if the core journey fails. Fix trust, explanation, upload, active report state, JD match clarity, and mobile completion first.

## Next Build Blocks

- Analytics/founder dashboard.
- Scoring calibration.
- Proof Engine v2.
- Roadmap execution system.
- Monetization readiness later.
