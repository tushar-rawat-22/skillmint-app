# Design Philosophy

SkillMint should feel intelligent before it feels beautiful.

Every interface element must earn its place.

The product should communicate clarity, confidence and momentum.

Never visual noise.

Never decoration for the sake of decoration.

# White Space

White space is a feature.

Never attempt to fill empty space.

Breathing room improves confidence.

---

# Grid System

12 Column Desktop

8 Column Tablet

4 Column Mobile

8px spacing system

Everything aligns to the grid.

---

# Layout Rules

Maximum content width

1440px

Dashboard content

Centered

Consistent margins

Large spacing between sections

Never edge-to-edge cards.

---

# Typography

Display

Large metrics

Headings

Section titles

Body

Readable

Supporting

Muted

Never use more than four font sizes on one screen.

---

# Card Philosophy

Every card answers one question.

Examples

Career IQ

↓

How employable am I?

Mission

↓

What should I do next?

Achievement

↓

What improved?

If a card answers two questions

Split it.

---

# Visual Hierarchy

Level 1

Today's Mission

Level 2

Career IQ

Level 3

AI Coach

Level 4

Recent Achievement

Level 5

Secondary Navigation

Analytics never outrank actions.

---

# Shadows

Very soft.

Used only to separate layers.

Never dramatic.

---

# Corners

Consistent.

No random border radii.

---

# Borders

Thin.

Low contrast.

Support structure.

Never dominate.

---

# Motion

Motion communicates

Progress

Success

Hierarchy

Navigation

Never entertainment.

---

# Charts

Charts are supporting content.

Never the hero.

Charts exist only when they answer a question.

---

# Icons

Simple.

Consistent stroke.

Minimal.

Never decorative.

---

# Buttons

Primary

One per screen.

Secondary

Low emphasis.

Danger

Rare.

Never compete visually.

---

# Empty Space Rule

If adding another card does not improve decision making

Do not add it.

---

# Dashboard Rule

The dashboard is not a report.

The dashboard is Mission Control.

---

# Share Philosophy

Achievements are designed to be shared.

Dashboards are not.

---

# Premium Rule

Users should immediately feel

"I paid for quality"

even before upgrading.

Premium is communicated through craftsmanship.

Not effects.

---

# The SkillMint Test

Every screen must pass three questions

Can it be simpler?

Can it be faster?

Can it create confidence?

If any answer is No

Redesign.
