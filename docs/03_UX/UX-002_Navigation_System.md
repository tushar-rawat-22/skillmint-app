# Navigation Principles

Every page has one destination.

Every destination has one purpose.

Every purpose has one CTA.

No dead ends.

No circular navigation.

No hidden features.

---

# UX Success Criteria

A first-time user should reach Career Workspace in under five minutes without external guidance.

An existing user should reach any major feature within two clicks.

The navigation should remain consistent across desktop, tablet, and mobile.
