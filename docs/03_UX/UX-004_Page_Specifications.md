# UX-004 — Page Specifications

Document ID: UX-004

Version: 1.0

Status: Approved

---

# Purpose

Defines the responsibility of every page.

Each page exists for exactly one primary purpose.

---

# P-001 Landing

Purpose

Build trust.

Primary CTA

Get Started

Success Metric

User creates an account.

---

# P-002 Login

Purpose

Authenticate users.

Primary CTA

Sign In

---

# P-003 Register

Purpose

Create account.

Primary CTA

Create Account

---

# P-004 Welcome

Purpose

Reduce anxiety.

Explain what happens next.

Primary CTA

Upload Resume

---

# P-005 Resume Upload

Purpose

Collect resume.

Primary CTA

Upload PDF

Success Metric

Resume successfully uploaded.

---

# P-006 AI Processing

Purpose

Increase anticipation.

Never show a static spinner.

Display:

• Parsing Resume

• Evaluating Skills

• Comparing Market

• Building Career IQ

Estimated Time

20–30 seconds.

---

# P-007 Career Reveal

Purpose

Deliver emotional impact.

The first screen should reveal:

Career IQ

↓

Short AI Summary

↓

One Key Strength

↓

One Biggest Opportunity

↓

Continue

No charts.

No clutter.

# P-011 Profile

# P-012 Settings

# P-013 Pricing

# P-015 404

Purpose

# Golden Rule

Every page must answer one question.

If it answers two questions,

split the page.
