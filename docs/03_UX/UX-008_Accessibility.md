# UX-008 — Accessibility

Document ID: UX-008

Version: 1.0

Status: Approved

---

# Purpose

SkillMint should be usable by everyone.

Accessibility is a product requirement.

Not an enhancement.

---

# Standards

Target

WCAG 2.2 AA

---

# Keyboard Navigation

Every interactive element

must be keyboard accessible.

Logical tab order.

Visible focus states.

No keyboard traps.

---

# Color

Never communicate meaning

through color alone.

Every status includes:

Icon

Label

Color

---

# Typography

Minimum readable font

16px

Adequate line spacing

High contrast

Avoid long paragraphs.

---

# Screen Readers

Every interactive element

requires descriptive labels.

Charts require textual summaries.

Images require meaningful alt text.

---

# Forms

Every field

has

Visible Label

Helper Text

Validation

Error Message

Required Indicator

---

# Error Messages

Explain

What happened

Why

How to recover

Never display technical jargon.

---

# Motion

Support

prefers-reduced-motion

Reduce non-essential animation.

---

# Touch Targets

Minimum

44x44 px

Adequate spacing

Avoid accidental taps.

---

# Contrast

Text

Minimum WCAG AA

Buttons

Clearly distinguishable

Focus rings always visible.

---

# Responsive Behaviour

Desktop

Tablet

Mobile

All layouts must preserve usability.

No horizontal scrolling.

---

# Charts

Every chart includes

Title

Description

Accessible summary

Meaning

Charts should never be the only way information is presented.

---

# AI Accessibility

Recommendations should be written

in plain language.

Avoid unexplained technical terminology.

Offer expandable explanations for advanced users.

---

# Language

Use

Clear

Simple

Action-oriented copy

Examples

Good

"Improve your ATS score."

Bad

"Optimization recommendation generated."

---

# Accessibility Checklist

Before release

✓ Keyboard tested

✓ Screen reader tested

✓ Mobile tested

✓ Contrast verified

✓ Focus order verified

✓ Reduced motion supported

✓ Error recovery verified

---

# Final Principle

Accessibility improves usability for every user.

It is part of the SkillMint experience.
