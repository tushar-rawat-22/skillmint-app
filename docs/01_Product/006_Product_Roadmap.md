# Purpose

This roadmap defines the planned evolution of SkillMint.

Dates are intentionally omitted.

Progress is milestone-driven rather than calendar-driven.

---

# Roadmap Principles

Features are prioritized according to:

1. User value
2. Product simplicity
3. Engineering complexity
4. Business impact
5. Long-term scalability

No roadmap item should compromise the stability of previous phases.
