# Purpose

This document defines the primary user groups SkillMint is designed to serve.

These personas guide UX decisions, feature prioritization, AI recommendations, and product strategy.

---

# Persona 1 — College Student

## Description

Students currently pursuing undergraduate or postgraduate education with limited industry experience.

## Goals

- Build employable skills
- Obtain internships
- Build a strong resume
- Understand industry expectations

## Frustrations

- Doesn't know what recruiters expect
- Weak portfolio
- No roadmap
- Information overload

## Primary SkillMint Features

- Career IQ
- Resume Analysis
- AI Roadmap
- Skills Radar
- ATS Score

---

# Persona 2 — Final Year Student

## Description

Students actively preparing for placements.

## Goals

- Become placement ready
- Crack campus interviews
- Improve resume
- Increase recruiter confidence

## Frustrations

- Low confidence
- Resume rejection
- Poor interview preparation

## Primary Features

- Resume Upload
- ATS Analysis
- Recruiter Confidence
- Salary Intelligence
- AI Suggestions

---

# Persona 3 — Early Career Professional

## Description

Professionals with 0–5 years of experience seeking career growth.

## Goals

- Switch companies
- Increase salary
- Identify skill gaps
- Improve market value

## Frustrations

- Doesn't know market expectations
- Resume outdated
- Unsure which skills to learn

## Primary Features

- Salary Intelligence
- Career IQ
- Resume Studio
- AI Recommendations

---

# Persona 4 — Career Switcher

## Description

Professionals transitioning into a different domain.

## Goals

- Learn required skills
- Build confidence
- Understand transition roadmap

## Frustrations

- Doesn't know where to start
- Unsure whether they're ready for realistic roles
- High uncertainty

## Primary Features

- Career Roadmap
- Skills Radar
- AI Recommendations
- Career IQ
