# Purpose

This document defines the exact scope of SkillMint Version 1.

Anything not listed here is considered out of scope unless approved through a future roadmap revision.

---

# MVP Goal

Enable a user to upload a resume, receive actionable AI-driven career analysis, and continuously improve employability through measurable guidance.

# Explicitly Excluded

- University Portal
- Team Accounts
- Browser Extension
- Mobile Apps
- Public API

- AI Voice Interview
- Community Features
- Messaging

---

# Definition of MVP Success

The MVP is successful if a user can:

1. Register
2. Upload a resume
3. Receive meaningful analysis
4. Understand current employability
5. Receive an improvement roadmap
6. Return later and monitor progress

---

# Scope Governance

No additional features shall be added to the MVP unless they satisfy all of the following:

- Solves an existing JTBD.
- Supports the SkillMint vision.
- Does not delay public launch.
- Receives formal approval.
