# SkillMint Premium UI QA

**Scope:** Beta v1 Block 1 - Premium Product UI System

## Visual QA

- [ ] Desktop: dashboard, resume, ATS, roadmap, settings, upload, setup, login, and signup feel premium, minimal, and light-toned.
- [ ] Tablet: two-column layouts remain readable and do not create awkward empty cards.
- [ ] Mobile: pages stack to one column without horizontal overflow.
- [ ] Buttons, badges, and filenames wrap safely on small screens.
- [ ] Touch targets are comfortable and not crowded.

## Dashboard QA

- [ ] Dashboard feels like a serious career report, not a neon dashboard.
- [ ] Career IQ and Proof Confidence are readable and clearly explained.
- [ ] Harsh Truth is visible, direct, and not cramped.
- [ ] Proof Evidence Breakdown has no empty grid cells or placeholder blocks.
- [ ] Report guide does not collapse all scores into one vague metric.
- [ ] Profile-fit roles remain visually and verbally separate from Latest JD Match.
- [ ] Dashboard restore latest saved report still works.
- [ ] Active report rules remain visible and truthful.

## Shareable Snapshot QA

- [ ] Shareable Snapshot looks like a premium career report card.
- [ ] Generated by SkillMint badge is visible.
- [ ] Share summary button works through native share or clipboard fallback.
- [ ] Career IQ is visually prominent.
- [ ] Profile-fit role is shown without "Best Match" language.
- [ ] Proof Confidence is shown when available.
- [ ] Latest JD Match is shown only when available.
- [ ] Next Proof Move is included.
- [ ] No job guarantee, fake verification, or fake image export is implied.

## Page QA

- [ ] Resume history is readable and shows Current active report clearly.
- [ ] Latest 4 non-active saved analyses show by default.
- [ ] Show older analyses / Show fewer still works.
- [ ] Repeated filenames are distinguishable by date/time.
- [ ] Long filenames wrap safely.
- [ ] ATS/JD page clearly says Latest JD Match is one pasted JD.
- [ ] Roadmap missions feel mature and career-focused, not childish or arcade-like.
- [ ] Settings clearly says Clear workspace is browser-only.
- [ ] Settings does not imply account deletion or synced history deletion.
- [ ] Upload and setup feel consistent with the premium product.
- [ ] Landing page is premium light-toned and readable across hero, preview, FAQ, CTA, nav, and footer.
- [ ] Landing page has no fake testimonials, fake ratings, fake logos, fake company names, or placement claims.
