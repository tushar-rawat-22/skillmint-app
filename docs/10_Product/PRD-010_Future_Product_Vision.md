# PRD-010: Future Product Vision

SkillMint's long-term direction is to become the trusted proof graph and placement readiness operating system for early-career talent.

## Near-Term

- Clearer active target selection
- Allow user to choose a profile-fit role
- Better profile-fit role explanations
- Feedback system improvements
- Beta analytics
- Report export and sharing

## Mid-Term

- GitHub proof scanning
- LinkedIn profile analysis
- LeetCode and coding profile signals
- Portfolio and project proof validation
- Public SkillMint profile
- Improved roadmap personalization
- Guided readiness sprints

## Long-Term

- Career proof graph
- Verified early-career talent profile
- Recruiter-facing proof profile
- College and institution dashboards
- AI career coach
- Placement readiness operating system
- SkillMint readiness credential

## External Proof Principle

External validation must be labeled clearly.

Until scanned and validated, links are evidence candidates, not verified proof.

SkillMint should never imply certainty before proof sources are inspected reliably.
