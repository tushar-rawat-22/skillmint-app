# PRD-009: Monetization And Offers

SkillMint should not position basic scoring as the main product.

The product sells career clarity, proof-building guidance, role readiness, JD readiness, structured career progress, and placement readiness intelligence.

## Free Beta

The free beta can include:

- Resume upload
- Basic Career IQ
- Proof Confidence
- Profile-fit roles
- Basic JD match
- Basic roadmap
- Feedback collection

Free users should feel real value before any paid prompt.

## Paid B2C Offers

### 1. Pro Career Report

A detailed report covering role-fit, proof gaps, JD readiness, and roadmap.

### 2. 30-Day Readiness Sprint

Guided weekly missions for one target role.

### 3. Proof Review

Review projects, GitHub, portfolio, LinkedIn, and resume bullets.

### 4. JD Application Pack

For one job:

- JD gap analysis
- Resume rewrite guidance
- Missing proof list
- Application readiness report

### 5. Interview Readiness Pack

Future:

- Role-specific questions
- Project defense
- Proof-based interview prep

## Paid B2B / B2B2C Offers

### 1. College Placement Readiness Dashboard

Aggregated student readiness, role clusters, and proof gaps.

### 2. Coaching Institute Dashboard

Batch-level career readiness and progress tracking.

### 3. Placement Trainer Toolkit

Reports and action plans for students.

### 4. Recruiter-Ready Talent Proof Profiles

Future product after external proof validation is reliable.

## Pricing Direction

- India-first affordable pricing.
- Start with reports and readiness sprints before subscriptions.
- B2B can become larger than B2C.

No payment integration is part of this document.
