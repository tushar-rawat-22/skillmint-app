# PRD-008: Beta User Journey

This document defines the beta journey SkillMint should optimize before public launch.

## 1. Landing

The user understands SkillMint is a career operating system for students and freshers.

The landing page should explain the core promise:

- Stop applying blindly.
- Know your proof gaps.
- Match real jobs.
- Build your next 30 days.

## 2. Signup / Login

The user creates an account or returns to continue progress.

The auth flow should feel premium, simple, and connected to the product promise.

## 3. Setup

The user provides intent:

- Career field
- Target role
- Goal
- Pace

Setup is user intent, not final truth. SkillMint should use it for guidance and framing, not as proof.

## 4. Upload Resume

SkillMint reads the user's current reality from the uploaded resume.

This begins the intelligence loop.

## 5. Resume Intelligence

SkillMint shows:

- Parsed resume
- Proof Confidence
- Base Resume Signals
- Evidence-backed claims
- Weakly supported claims
- Unverified claims

The page should make clear that base signals are detection scores, while Proof Confidence is the trust layer.

## 6. Dashboard

The dashboard shows:

- Career IQ
- Active Target / Latest JD
- Proof Confidence
- Next Best Action
- Proof Evidence
- Profile-fit roles
- Roadmap direction

The dashboard should feel like the command center, not an admin report.

## 7. ATS / Latest JD Match

The user tests one real job description.

SkillMint shows:

- Match score
- Missing skills
- Keyword gaps
- Rewrite suggestions
- Readiness warning

Latest JD Match answers whether the user can compete for that exact job. It does not replace Profile-fit Roles.

## 8. Roadmap

SkillMint gives a 30/60/90-day plan for the Active Target.

If a latest JD exists, roadmap guidance should follow that role context first.

## 9. Feedback

During beta, user feedback should validate:

- Did SkillMint help them understand what roles to apply for?
- Did SkillMint make gaps clear?
- Did SkillMint tell them what to build next?
- Did SkillMint feel trustworthy?
- Was anything confusing?
